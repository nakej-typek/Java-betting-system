# BetCore

Semester school project.

A Java domain model of a betting office: sports competitions (matches, races, groups, knockout brackets), betting markets with odds history, bettors, result evaluation and JSON persistence.

## Structure

```
app/         runnable demos
model/       domain layer (bet, bettor, competition, outcome, participant, result)
services/    application logic and JSON repository
docs/        UML class diagrams (PlantUML)
```

## Tech

Java 25, Maven, JUnit 5, Jackson, Checkstyle.
